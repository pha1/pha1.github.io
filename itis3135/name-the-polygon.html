Name the Polygon

Welcome to our polygon page! Here are some things you should know about polygons:

  • A polygon is traditionally a plane figure that is bounded by a finite chain of straight line segments closing in a loop to form a closed chain.
  • Those segments are called edges or sides.
  • The points where the edges meet are called vertices (singular: vertex).

If you would like to know the name of a polygon with a specific number of sides, please press the following button to begin!

This program allows you to enter a number of sides for a polygon (up to 19!) and returns the name of the polygon!
